## 注册

  1. 获取注册页面路由
  > get /signup

  2. 注册请求接口
  > post /signup

  ```js
    //请求参数
    {
      username:'xxx',
      password:'xxx',
      sex:'',
    }
    //响应参数
    {
      status:'xxx',
      msg:'xxx'
    }
  ```
